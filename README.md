# collabstudy

A collaborative study room application built using React and Firebase.

## Features
- User authentication
- Create & join study rooms
- Task and deadline management
- Pomodoro timer
- Study Tracker (new feature)

## Study Tracker
- Tracks study activity inside each room
- Integrated into the Room page
- Foundation for future analytics

## Tech Stack
- React + Vite
- Firebase (Auth + Firestore)
- Tailwind CSS
